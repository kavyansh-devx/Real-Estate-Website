# EstateX AI Real Estate

A luxury real estate portfolio website built with React, TypeScript, Vite, Tailwind CSS, Framer Motion, Zustand, and Three.js.

## Highlights

- Cinematic animated homepage with rotating hero imagery
- Curated property collection with filters and saved wishlist state
- Property detail pages with gallery controls, amenities, and contact CTAs
- AI-style concierge chat experience
- Consultant directory with direct email and phone actions
- Validated contact form with polished success and error states
- Responsive layout for desktop, tablet, and mobile

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Local Preview

```bash
npm run dev
```

Then open the local Vite URL shown in the terminal.
